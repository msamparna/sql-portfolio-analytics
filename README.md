# SQL Portfolio Analytics Queries

A collection of SQL queries for portfolio data analysis, financial reporting, and KPI tracking — built to reflect real-world use cases in investment management and operations.

## 📁 Query Categories

| File | Description |
|---|---|
| `portfolio_performance.sql` | Returns, NAV, and benchmark comparison |
| `kpi_tracking.sql` | KPI monitoring across business units |
| `variance_reporting.sql` | Actual vs budget variance analysis |
| `vendor_ops.sql` | Vendor performance and spend analysis |

## 🛠️ Tools Used
- SQL (MySQL / PostgreSQL compatible)
- Designed for integration with Power BI and Excel
